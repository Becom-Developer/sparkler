## DEPLOY

```
(鍵認証による接続 ローカルマシンより)
$ ssh becom@becom.sakura.ne.jp

(もしくは)
$ ssh becom@sparkler.becom.work

(さくらのレンタルサーバーは csh)
% pwd
/home/becom

(移動後 git を最新の状態に)
% cd www/sparkler/
% pwd
/home/becom/www/sparkler

% git fetch
% git pull origin master
```
